Web 知识

HTTP 请求方式

HTTP 是一种协议，定义了客户端和服务端的请求规范。HTTP 默认使用 80 端口指向服务，而 HTTPS 协议的默认端口为 443。

HTTP 有下表中的八种方法：

方法 描述
GET 请求获取指定页面的实体数据
HEAD 请求获取指定页面的头信息
POST 向指定资源提交数据进行数据请求
PUT 传输数据取代指定文档的内容
DELETE 请求删除指定页面
OPTIONS 允许查看服务器性能
TRACE 返回服务器收到的请求
